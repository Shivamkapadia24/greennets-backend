GreenNets E-commerce Backend:

This is the complete MERN stack backend for my GreenNets e-commerce project. It includes:
User authentication with JWT
Product, Cart, and Wishlist management (CRUD)
AI Chatbot integration with Google Gemini

To Run This Project:
1. npm install

2. Create a .env file and add the following variables:
MONGODB_URI=
JWT_SECRET=
GEMINI_API_KEY=

3. npm run seed (to populate the database)

4. npm start
